# Retirement Calculator (Static + Pyodide)

This static retirement planner projects savings growth and estimated retirement income.

## Features
- Inputs include age, savings, contributions, return, inflation, withdrawal rate
- Outputs projected balance and monthly income in nominal and today's dollars
- Optional target income check with on-track indicator
- Copy Results and responsive design

## Run locally
```bash
cd retirement-calculator
python3 -m http.server 8000
```
Open `http://localhost:8000`.
